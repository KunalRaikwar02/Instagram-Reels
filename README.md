# 🎬 Reels App (Instagram Reels Clone)

A dynamic and interactive reels application built using HTML, CSS, and JavaScript.  
Users can scroll through videos, like, follow, mute, and interact just like a real social media platform.

---

## 📌 Overview

This project is a front-end clone of Instagram Reels.

It demonstrates:
- Dynamic rendering using JavaScript
- State management (like, follow, mute)
- Video autoplay/pause using Intersection Observer
- Interactive UI with real-time updates

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- Remix Icons  

---

## ✨ Features

- 📱 Vertical scroll reels (snap scrolling)
- 🎥 Auto play/pause videos on scroll
- ❤️ Like / Unlike functionality
- 👤 Follow / Unfollow users
- 🔇 Mute / Unmute toggle
- 💬 Comment & Share UI
- ⚡ Dynamic rendering using JS
- 🎯 Real-time UI updates
- 🔥 Smooth scrolling experience

---

## 🧠 Key Concepts Used

- DOM Manipulation  
- Event Delegation  
- Intersection Observer API  
- State Handling (Objects & Arrays)  
- Dynamic HTML Rendering
